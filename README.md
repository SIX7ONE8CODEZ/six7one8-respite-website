# SIX7ONE8 RESPITE Website

Professional respite care services website for SIX7ONE8, LLC.

## About

SIX7ONE8 RESPITE provides comprehensive respite care services including:

- **Community Connector** - Youth services up to age 17 for community integration
- **Homemaker Services** - Basic and enhanced household support
- **Mentorship** - Self-advocacy and independence training programs  
- **Respite Care** - Individual, day, and group respite options
